Usage: z88dk-z80asm [options] files...
Copyright (C) Paulo Custodio, 2011-2026

Assembles Z80 assembly source files into binary output.

Help Options:
  -h            Show this screen
  -v            Verbose output

File and Directory Options:
  -I=DIR        Add directory to search for source/include files
  -O=DIR        Set directory to store output files

External Preprocessor Options:
  -cpp=OPTS     Pass options to cpp
  -m4=OPTS      Pass options to m4
  -perl=OPTS    Pass options to perl

Preprocessor Options:
  -IXIY         Swap IX and IY registers
  -ucase        Uppercase symbols
  -D=NAME[=EXPR]Define symbol
  -E            Preprocess only

Assembly Options:
  -m=CPU        Set target CPU (e.g. z80, z180)
  -d            Assemble only if newer

Diagnostic Options:
  -dump-after-cmdline
                Dump the command line arguments and exit
  -dump-after-tokenization
                Dump the lexer token stream and exit
  -dump-after-directives
                Dump the token stream after directive processing and exit
  -dump-after-macro-expansion
                Dump the token stream after macro expansion and exit
  -dump-after-preprocessing
                Dump the token stream after preprocessing and exit
  -dump-after-hla
                Dump the token stream after High-Level-Assembly processing and exit
